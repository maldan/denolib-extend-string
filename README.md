Extend standard js string with this module.

## Full documentation

Go here - https://doc.deno.land/https/deno.land/x/estring/mod.ts

## How to import

```ts
import { EString } from "https://deno.land/x/estring@1.0.0/mod.ts";
```

## Example

```ts
EString("hello").capitalize(); // Hello
```
